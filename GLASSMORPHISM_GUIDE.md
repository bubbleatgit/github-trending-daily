# 毛玻璃风格 GitHub Trending 界面

## ✨ 新增文件

我已经为你创建了毛玻璃风格（Glassmorphism）的全新界面，包含以下文件：

1. **index-glass.html** - 毛玻璃风格的主页面
2. **style-glass.css** - 毛玻璃风格的CSS样式
3. **app-glass.js** - 适配毛玻璃风格的JavaScript逻辑

原有的 `index.html`, `style.css`, 和 `app.js` 文件保持不变，你可以随时切换回原版本。

## 🎨 毛玻璃风格特点

- **半透明背景** - 所有卡片使用 `rgba(255, 255, 255, 0.08)` 半透明背景
- **模糊效果** - 使用 `backdrop-filter: blur(20px)` 创建毛玻璃效果
- **柔和边框** - 使用 `rgba(255, 255, 255, 0.18)` 轻柔的白色边框
- **渐变背景** - 精美的紫色/蓝色渐变背景
- **悬浮动画** - 卡片悬停时平滑上浮和阴影增强
- **现代配色** - 深色主题配合鲜艳的强调色
- **渐变文字** - 关键数据使用渐变文字效果
- **平滑动画** - 所有交互都有流畅的过渡动画

## 🚀 如何使用

### 方法一：预览本地效果

你可以直接在浏览器中打开 `index-glass.html` 文件查看效果：

```bash
# 在文件管理器中双击 index-glass.html
# 或使用浏览器打开
```

### 方法二：部署到 GitHub Pages

1. 将以下文件上传到你的 GitHub 仓库：
   - `index-glass.html`（替换原有的 index.html）
   - `style-glass.css`
   - `app-glass.js`

2. 建议保留原文件备份，便于切换

### 方法三：保留双版本

如果你想同时保留两个版本：
- 原版：`index.html` + `style.css` + `app.js`
- 毛玻璃版：`index-glass.html` + `style-glass.css` + `app-glass.js`

访问时选择对应的 HTML 文件即可。

## 🔧 自定义颜色

如果需要调整配色，编辑 `style-glass.css` 的 `:root` 部分：

```css
:root {
    --primary: #60a5fa;          /* 主色调 */
    --success: #34d399;           /* 成功/增长色 */
    --gradient-1: #667eea;       /* 渐变起始色 */
    --gradient-2: #764ba2;       /* 渐变结束色 */
    /* ... 其他变量 */
}
```
